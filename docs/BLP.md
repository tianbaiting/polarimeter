# BLP（束流极化计）


## 核心原理
BLP 的核心原理是利用已知分析能力（Analyzing Power, A_y）的核散射反应（通常为质子-质子 p-p 或氘核-质子 d-p 弹性散射）来推算束流极化度。

- 不对称性测量：极化束流打靶时，左右（或上下）散射粒子计数产生不对称性，通过测量计数率差异提取极化信息。  
- 符合测量（Kinematic coincidence）：为消除非弹性散射或碎裂反应等背景，通常要求同时探测前向散射粒子和反冲粒子，且在时间和运动学角度上符合才记为有效事件。

## 探测器结构配置
BLP 的探测器系统一般由多对塑料闪烁体计数器（Plastic Scintillation Counters）及光电倍增管（PMT）构成。

- 对称布局：探测器在垂直于束流的平面上对称放置，分为左（Left）、右（Right）、上（Up）、下（Down）四个方位。  
- 多功能用途：左右两对（L–L' 和 R–R'）主要测量水平方向极化不对称性；上下两对（U–U' 和 D–D'）用于特定极化分量测量及监测束流强度或传输效率。  
- 特定角度：为获得最佳分析能力，探测器布置在 A_y 较大的角度。例如对约 300 MeV 的质子束，实验室系散射角常设为 17.0°，反冲角约 70.5°（或 69.7°），此时 A_y ≈ 0.40。

## 靶材的选择与“抗辐照”问题
靶材稳定性对连续测量极化度至关重要，相关对比研究如下：

- 聚乙烯薄膜（CH2）：富含氢且分析能力已知，是传统 BLP 靶材。但在强束流长时间照射下氢含量会下降，需通过移动打靶位置等方法恢复测量效率。  
- 芳纶薄膜（Aramid）：如日本旭化成生产的芳纶（近似成分 C14O2N2H3）在累计电荷约 100 µC 内氢丰度稳定，无明显流失。基于 CH2 的标定，295 MeV 质子束下芳纶靶的 A_y 被标定为 0.38 ± 0.01。

## 各大实验室的具体 BLP 系统实例

- RCNP（大阪大学核物理研究中心）：在 WS 高分辨率束流线上串联配置两个 BLP（BLP1 与 BLP2）。一次长期实验中，BLP1 使用 5 mg/cm² 芳纶薄膜，BLP2 使用 8.4 mg/cm² CH2 薄膜以相互校验。  

https://www.rcnp.osaka-u.ac.jp/Divisions/np1-a/ws/index.html

RCNP（大阪大学核物理研究中心）在其设施内制作并安装了束流极化计（BLP）。参考：[Matsubara (2005) 学位论文](https://www.rcnp.osaka-u.ac.jp/Divisions/np1-a/thesis/Matsubara_master2005.pdf)

- RIKEN（日本理化学研究所）：在能量 140–270 MeV 极化氘核实验中使用两套 BLP：一套（D-room 极化计）位于回旋加速器下游用于加速后极化度初步确认；另一套（swinger 极化计）位于散射室前方，随束流偏转磁铁移动以测量打靶前的瞬时极化度。  



- JINR（俄罗斯 Dubna）：在 Nuclotron 加速器环的内部靶站（ITS）实现高精度测量氘核张量极化分量（tensor polarization）。
